# **App Name**: JSON Portfolio

## Core Features:

- Content Storage: Store portfolio content (projects, essays, reviews, memos) in structured JSON files within the GitHub repository.
- Navigation: Implement navigation to Home, Projects, Essays, Reviews, Memos, and CV & Contact pages.
- Home Page: Display intro text and the latest items from each content category.
- Category Pages: List items from corresponding JSON files, showing thumbnail, title, intro, and date; filter by tags.
- Post Pages: Display full content with banner image, title, subtitle, intro, date, and clickable tags. Reviews display star rating.
- Client-Side Search: Implement a client-side search in plain JavaScript to match by title, tags, date, and full text content.
- CV & Contact: Embed CV iframe with PDF download option and include a contact form with basic spam protection (honeypot or reCAPTCHA).

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) for a professional and sophisticated feel.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop.
- Accent color: Teal (#008080) for highlighting key elements and CTAs.
- Body font: 'PT Sans', sans-serif, for a modern yet approachable look.
- Headline font: 'Space Grotesk', sans-serif, for a computerized, techy, scientific feel. For longer text, use 'Inter' for body.
- Use a set of simple, professional icons to represent different content categories and tags.
- Responsive design to ensure optimal viewing experience across devices. Clean URLs for SEO.
